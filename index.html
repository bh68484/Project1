Have you ever wondered if your favorite NFL player was a stand up guy? Now you can find out.

The application allow you easily display arrest information along with a photo of the player. Users can quickly add multiple players to the screen and can compare arrest information between them. Using the application is straightforward.

We divided the workload with Colin and Kross focusing on the front-end.

Brandon and Matt focusing on the back-end.

Front-end was created using materialize framework

Back-end was built by using two APIs. The first API used was NFL Arrest API which was used to search a player's name to see if they were found in the database.
